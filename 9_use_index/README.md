# Índices

## Síntaxe Básica de um index
* __Basicão:__

    CREATE INDEX nome_do_index 
        ON tabela (coluna);

* __Índice Único:__

    CREATE UNIQUE INDEX nome_unico_do_index ON tabela (coluna);


* __Índice para múltiplas colunas:__

    CREATE INDEX nome_index_colA_colB ON tabela (colA, colB);
